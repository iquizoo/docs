基本信息

  • 显示名称：火眼金睛
  • 英文名称：SearchObject
  • 题目编号：03703
  • 任务描述：测查注意力，特别是从大量物体中查找出所需物体的能力。
  • 测查能力：注意力-选择注意
  • 相关任务：视觉搜索、注意搜索

前端交互

以梅鹏提供的为基础。

后台配置

游戏参数

目前不设置配置参数。

数据记录

表1: 原始数据变量列表
名称 类型 说明
NCand int 备选物体数目，目前定为30
STIM char 需查找的物体名称
Resp char 用户的实际选择的物体
ACC int 用户作答是否正确，1=正确，0=错误
RT int 用户作答所用时间（单位：毫秒）

试题算分

表2: 传入参数（算分码：203）
参数名称 类型 说明
rightCnt int 作答正确的个数
wrongCnt int 作答错误的个数
allTime int 完成测评的累计消耗时间，单位：毫秒

算分方法

采用一个考虑了速度准确性权衡的指标。类似于RCS。规则：

\[分数 = \frac{作答正确的个数}{整个测验的持续时间}\]

注意整个测验的持续时间就是给用户的作答限时，且使用分钟作为单位。
