  • 显示正确判断的个数

数据提交

表1说明了对于每一个试次需要记录的数据（请参考标准）。

表1: 原始数据变量列表
名称 类型 说明
SR int 红色点的个数
SB int 蓝色点的个数
CResp char 正确答案。两种可能，即“Red”（红色）、“Blue”（蓝色）。
Resp char 用户的实际反应(Response)。三种可能，即“Red”（红色）、“Blue”（蓝色）、“None”（用户未操作）。
ACC int 正确或错误(ACCuracy)。0=错误，1=正确，-1=未完成操作。
RT int 用户完成操作所需时间（单位：毫秒）(Response Time)，若未完成操作，则记为0
