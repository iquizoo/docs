基本信息

  • 显示名称：思维转换
  • 英文名称：Shifting
  • 题目编号：03201
  • 任务描述：测查思维力，特别是灵活切换思维方式的能力。
  • 测查能力：思维力-思维灵活性
  • 相关任务：反应转换。本题目是反应转换的简化版本，推荐应用于1-4年级。

交互界面

每一个试次中，屏幕上呈现一张背景图片（即线索，背景图片中央有空白，用于显示动物图片），一定时间（线索目标间隔）后屏幕中央呈现一张动物图片（即目标），背景图片和动物图片会一直停留在屏幕上，直到用户需根据背景图片对屏幕中央的图片做出操作（或达到最大展示时间）后消失。最后，屏幕保持一定时间（反应线索间隔）的空白后进入下一个试次。

练习。本题目的背景图片有两种（参看详细说明部分），意味着实际上此题目包含了两个子任务，且每一个试次会要求用户根据程序给定的一个子任务进行反应（这也是思维转换名字的由来）。因此，练习由三部分组成（详细参看详细说明部分）：

  1. 指引和练习第1个子任务，并保证用户理解第1个子任务。
  2. 指引和练习第2个子任务，并保证用户理解第2个子任务。
  3. 指引和练习在两个子任务混合起来（实际测试时候的形式）的时候如何操作。

详细说明

  • 刺激
    • 背景图片两张：其意义在告知用户当前试次需要执行的子任务。
      • 其一由左侧海洋、右侧陆地组成，此背景图片要求用户判断屏幕中央的图片所代表动物的生活环境(Habituation)。
      • 另一由左侧猪、右侧老鼠的天平组成，此背景图片要求用户判断屏幕中央的图片所代表动物的体重(Weight)跟猪还是老鼠更接近。
    • 动物图片（见下表）
    • 刺激呈现
      • 背景图片先于动物图片呈现，间隔时间为线索目标间隔，由配置参数cti（默认为500ms）控制
      • 动物图片呈现后背景图片不消失
      • 动物图片的最大展示时间由配置参数maxSt（默认为3000ms）控制
      • 用户对图片做出操作反应或达到最大呈现时间后，屏幕需在下一个试次开始前（即下一张背景图片出现前）变成空屏状态，持续时间为反应线索间隔，由配置参数rci（默认为500ms）控制。

  • 操作
    • Android/iOS：点击背景图片的左侧或右侧做出对应的反应。例如，在“左侧海洋右侧陆地”背景图片情况下，如出现“老虎”图片，用户需点击屏幕右侧（即“陆地”一侧）。
    • PC/Web：若PC拥有触摸屏，则用户可以使用Android/iOS的操作方式；若PC没有触摸屏，则要求用户使用标准输入设备键盘上的←和→进行操作，分别表示左侧和右侧。
  • 指引和练习
    • 指引和练习分为3个部分：
      1. 指引和练习生活环境子任务。背景图片依然呈现。至少完成8次，即至少每种目标词语做1次；8次后若正确率不高于80%，则继续练习至正确率高于80%或总计练习16次时结束。注意每个8次练习中都是将所有刺激按随机顺序呈现的（后面的练习也是如此）。
      2. 指引和练习判断体重子任务。背景图片依然呈现。至少完成8次，即至少每种目标词语做1次；8次后若正确率不高于80%，则继续练习至正确率高于80%或总计练习16次时结束。
      3. 指引和练习在两个子任务混合条件。提醒用户注意背景图片确定操作方式。此部分至少包含16个试次，保证所有刺激在两种子任务情况都分别练习1次，采用完全随机的顺序即可。16次后若正确率不高于80%，则继续练习至正确率高于80%或总计练习32次时结束。注意每个16次练习中都是将所有刺激在两种子任务情况都分别随机顺序地练习1次。
  • 序列
    • 一般要求
      • 相同的刺激必须至少间隔两次才能出现。
    • 特殊要求
      • 切换和重复的条件最多连续3次，且连续出现次数1, 2, 3的比例为1:2:1。注意：当重复条件出现\(n\)次时，会重复执行一个任务\(n + 1\)次。
      • 正确操作的一侧最多连续3次，且连续出现次数1, 2, 3的比例为1:2:1。
      • 保证任务类型、切换条件、正确操作相互均衡。
      • 保证刺激在不同的任务类型中比例一致。
  • 序列配置（参考标准中关于格式的说明）
    • 注意：本题目的第一个试次是一个“填充”试次，因此若设试次数目为\(n\)时，实际需要完成的试次数目为\(n + 1\)。在序列的配置标准中也是如此，即length为\(n\)序列中，stim个数都是\(n + 1\)。
    • 数组中每一个元素都是一个json对象，包含六个名称/值(name/value)配对，分别是
      • length：表示序列的长度
      • stim：包含了由刺激编号（和“动物图片及其属性一览表”一致）组成的多个序列
      • task：包含了由子任务名称组成的多个序列，其中Habituation表示判断动物的生活环境，Weight表示判断动物的体重大小
      • cond：包含了由任务条件，即任务是否切换，组成的多个序列，且每个序列中，第一个注明为Filler，其余为Repeat和Switch中的一个
      • cresp：包含了由正确答案组成的多个序列，可能值为Left和Right

表3: 原始数据变量列表
名称 类型 说明
Round int 本任务有轮次的概念，此处记录当前在第几个轮次
STIM int 刺激编号(STIMulus)，1-8中的一个
