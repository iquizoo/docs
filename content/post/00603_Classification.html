数据记录

表1: 原始数据变量列表
名称 类型 说明
STIM char 刺激名称（STIMulus）。
CResp char 正确反应（Correct Response），Left（左）或者Right（右）。
Resp char 用户的实际反应（Response），Left（左）、Right（右）或者None（未操作）。
ACC int 正确或错误(ACCuracy)，0=错误，1=正确，-1=未完成操作。
RT int 用户完成操作所需时间（Response Time，单位：毫秒），若未完成操作，则记为0

表2: 传入参数（算分码：200）
参数名称 类型 说明
average int 正确操作试次的平均反应时间，单位：毫秒
rate float 整体作答正确率，未作答算作错误
