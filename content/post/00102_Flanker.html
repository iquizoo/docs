基本信息

  • 显示名称：方向达人
  • 英文名称：Flanker
  • 题目编号：00102
  • 任务描述：测查自控力，特别是排除干扰的能力。
  • 测查能力：自控力-控制冲动
  • 相关任务：方向达人。本题目与该题目属于同种题目，且本题目适用于10岁及10岁以下儿童。

后台配置

数据记录

对于每一个试次，需要提交以下数据：

表1: 原始数据变量列表
名称 类型 说明
Round int 本任务有轮次的概念，此处记录当前在第几个轮次
STIM int 刺激编号(STIMulus)，1-4中的一个
Type char 刺激类型，包括2种可能：‘Congruent’, ‘Incongruent’
CResp char 正确的反应(Correct Response），包含两种可能：‘Left’, ‘Right’
Resp char 用户的实际反应(Response)，包含两种可能：‘Left’, ‘Right’, ‘None’（未操作）
ACC int 正确或错误(ACCuracy)，0=错误，1=正确，-1=未完成操作。
RT int 用户完成操作所需时间（单位：毫秒）(Response Time)，若未完成操作，则记为0

试题算分

表2: 传入参数（算分码：201）
参数名称 类型 说明
medRT int 用户在“不一致条件”(Incongruent)下的反应时间的中位数
ACC float 用户在整个题目中的正确率

算分方法

参考美国国立卫生院的做法。规则：

\[score = PC + \frac{log_{10}(3000) - log_{10}(RT_{inc})}{log_{10}(3000) - log_{10}(500)}\]

其中\(PC\)是用户整体的正确比例（未作答记为错误），\(RT_{inc}\)是用户在不一致条件正确作答时的反应时间的中位数，且以毫秒为单位。

原始分的取值范围为0~2。
