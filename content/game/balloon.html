基本信息

  • 显示名称：速算大师
  • 英文名称：balloon
  • 题目编号：78
  • 任务描述：根据算术法则快速计算，训练更新加工信息的能力。
  • 相关任务：速算师（交互式题目）、速算大师（C2游戏）。

交互界面

以现有的C2游戏速算大师为准。需作如下调整：

  1. 修改气球大小适配两位数的显示。
  2. 修改选项数目为四个。
  3. 增加一个明显标志说明目前是做加法还是减法。
  4. 增加计分牌，显示目前得了多少分。（计分法则见详细说明）

详细说明

数字。使用两位数做素材（参考表1中的样例）。

题库配置。本题目提供一个表格文件，其中包含了所有可能的题目组合。表1给出了随机取出10个题目组合，其中：

  • left
    左侧数字
  • right
    右侧数字
  • type
    计算法则。add：加法，minus：减法。
  • level
    计算难度。easy：简单，medium：中等，difficult：困难。
  • score
    答对题目后得分数目。
  • correct, opt1, opt2, opt3
    备选选项。注意correct是正确答案，其他三个都是混淆的错误选项。实际呈现的时候，四个选项的位置是随机的。
表1: 题目组合样例
left right type level score correct opt1 opt2 opt3
73 45 add medium 2 118 117 128 108
14 60 add easy 1 74 75 84 64
13 87 add medium 2 100 99 110 90
67 52 minus medium 2 15 14 25 15
20 37 add easy 1 57 56 67 47
90 49 minus difficult 3 41 42 51 51
90 62 minus difficult 3 28 27 38 38
84 98 add difficult 3 182 181 192 172
60 94 add medium 2 154 155 164 144
37 18 add medium 2 55 54 65 45

关卡。只做3个关卡，用户依次完成。第一个关卡只做两位数的加法，第二个关卡只做两位数的减法，第三个关卡则按照“加法-减法-加法-减法-…”的顺序来做。每一个关卡都限时3分钟，同时每一组新的数字都是根据对应的计算法则从题库里面不重复地随机取出。

计分规则。用户起始时都为0分。对于每一道题目：

  • 若用户回答正确，则获得表1中设定的题目得分。
  • 若用户回答错误，则扣除1分。
  • 若用户未作答，则扣除3分。

数据记录

表2说明了对于每一对数字需要记录的数据（请参考标准）。

表2: 原始数据变量列表
名称 类型 说明
Left int 左侧数字
Right int 右侧数字
Type char 计算法则。add或minus，同题库配置
Level char 计算难度。easy、medium或difficult，同题库配置
CResp int 正确答案
Resp int 用户选择的选项。若未选择，则记为0
ACC int 是否正确。正确为1，错误为0，未作答为-1
RT int 作答时间。以毫秒为单位
