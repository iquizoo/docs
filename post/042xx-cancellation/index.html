  • 后台配置

基本信息

  • 显示名称：语言划消测验
  • 英文名称：cancellation
  • 题目编号：042XX
  • 任务描述：划消测验包含6个经典语言相关的任务，即符号判断（04201）、字形判断（04202）、声调判断（04203）、词语判断（04204）、语义判断（04205）和拼音判断（04206）。
  • 测查能力：语言/反应力-言语速度

后台配置

数据记录

表1: 原始数据变量列表1
名称 类型 说明
STIM char/int 刺激或刺激的编号
CResp char 正确答案，Left=按左键/触摸左侧按钮，Right=按右键/触摸右侧按钮
Resp char 用户的实际按键，Left=按左键/触摸左侧按钮，Right=按右键/触摸右侧按钮，若未作答记录为None
ACC int 用户作答是否正确，1=正确，0=错误，-1=未作答
RT int 用户作答所用时间（单位：毫秒）(Response Time）
