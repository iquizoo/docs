基本信息

  • 显示名称：联系记忆
  • 英文名称：faceSnack
  • 题目编号：02201
  • 任务描述：测查记忆力，特别是个体记住事物之间的联系（如人脸和名字）的能力。
  • 测查能力：记忆力-联系记忆

详细说明

正式测评需要修改为两轮，即把现在的学习和测试重复一次，要求：

  • 和现有的轮次学习相同的15个配对，但是出现顺序重新随机；
  • 测试时顺序也重新随机，并且修改选项。

结果显示

  • 显示两轮测试总共正确记忆出来的配对数目
  • 展示标准分数和超越百分比

数据提交

表1说明了对于每一个试次需要记录的数据（请参考标准）。

表1: 原始数据变量列表
名称 类型 说明
Round int 当前的轮次数目，1或者2。
Face int 人脸的编号。
Food char 备选零食的编号。使用“|”分隔不同零食，例如“12|3|9|16”。
CResp int 正确零食的编号。
Resp int 用户的实际选择的零食的编号。若用户未完成操作，则记为-1。
ACC int 正确或错误(ACCuracy)。0=错误，1=正确，-1=未完成操作。
RT int 用户完成操作所需时间（单位：毫秒）(Response Time)，若未完成操作，则记为0
