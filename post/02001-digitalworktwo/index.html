数据记录

表1: 原始数据变量列表1
名称 类型 说明
Round int 本任务有轮次的概念，此处记录当前在第几个轮次。
STIM int 当前试次呈现的数字（STIMulus）。
Type char 刺激的类型，有两种可能，即Filler（每轮的前两个试次）或者Target（每轮前两次之后的试次）。
CResp char 正确的反应（Correct Response），有两种可能，即Stay（不变）或者Change（变了）。
Resp char 用户的实际反应（Response），有三种可能，即Stay（不变）、Change（变了）或者None（未操作）。
ACC int 正确或错误（ACCuracy），0=错误，1=正确，-1=未操作。
RT int 用户完成操作所需时间（Response Time，单位：毫秒）
