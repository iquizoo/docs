基本信息

  • 显示名称：保卫地球
  • 英文名称：ProtectEarth
  • 题目编号：08501
  • 任务描述：测查思维力，特别是面对复杂问题进行深入思考并作出解答的能力。
  • 测查能力：思维力-思维深刻性
  • 相关任务：暂无

前端交互

后台配置

游戏参数

目前不设置配置参数。

数据记录

对于每一次陨石对地球的冲击，需记录以下数据：

表1: 原始数据变量列表
名称 类型 说明
Number int 当前陨石的顺序
Hit int 当前陨石是（1）否（0）会击中地球
ACC int 用户是（1）否（0）成功避免当前陨石集中地球

试题算分

表2: 传入参数（算分码：203）
参数名称 类型 说明
rightCnt int 作答正确的个数
wrongCnt int 作答错误的个数
allTime int 完成测评的累计消耗时间，单位：毫秒
